# Hyponatremia Navigator

互動式低血鈉症診斷、鑑別診斷與治療流程工具。

使用者需要逐步填入臨床症狀、血清與尿液檢驗、體液狀態、用藥、影像/病史資料，完成每一步必填資料後才會自動進到下一步，最後產生診斷分類、鑑別方向與治療警示摘要。

## 本機開啟

直接開啟 `index.html` 即可使用。

## 內容重點

- 低血鈉嚴重度與急慢性分類
- 非低張性/偽性低血鈉排除
- urine osmolality 與 urine sodium 導向式鑑別
- hypovolemic/euvolemic/hypervolemic 流程
- SIADH、CSW、HF、低溶質攝取、利尿劑與內分泌原因
- hypertonic saline、限水、tolvaptan/Samsca 與過度校正預防

## 注意

此工具為教學與臨床流程整理用途，不能取代醫師判斷、院內治療規範或即時病人照護。
